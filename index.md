# Installation

Öffne einen Pfad, in dem du das Repo speicherst.
Falls er nicht exestiert schreibe den Befehl `mkdir <Dein Pfad für das Repo>` und führe cd anschließend mit dem gleichen Argument auf

    cd <Dein Pfad für das Repo>

Lade das Repo herunter und gehe dahin:

    git clone https://github.com/viktor-chiarcos/vpackage
    cd vpackage

Installiere vpackage mit folgendem Befehl:

    ./install

# Verwendung

Um die Hilfe anzuzeigen verwende einfach den Befehl:

    vpackage help

# Hinweis

Diese Installation gillt nur für dein Benutzerkonto

Um den Namen von deinem Konto abzurufen verwende

    echo $USER
